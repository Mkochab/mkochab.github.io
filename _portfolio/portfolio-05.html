Catrine

Description

This is a 3D puzzle game involving physics-related puzzle mechanism. A punk cat is trapped in the limbo of cat litters due to its bad personal hygiene habit. We simulate litter particles in real time. The game has a voxelized art style.
My roles: story writter, program for rendering and physics, level design and mechanism program

SandBox

Description

We made an interactive digital sandbox with Microsoft Kinect2 and Unity. With Kinect, we bring in easy gesture controls. The scene has a terrain generated from height map and texture. The scene can be edited by users in run-time to create a customized environment. Then we can use the sand box as a playground. You can either fly a glider with gestures or command the toy soldiers and tanks to simulate a mini battlefield.
