Cloud generation

Procedurally generated clouds

A common method of cloud generation is introduced in [1] used in Horizon: Zero Dawn The cloud is procedurally generated using noise.

Physically-base generation

The real-time generation of clouds is based on several theoritical models: atmospheric model, fluid dynamics, and the water transition model of the clouds. . The implementation is mainly based on [2] and also adopt a few details from [3]. We can generate the clouds based on the input of heat and vapor emission, a set of control parameters, as well as the terrain information.
